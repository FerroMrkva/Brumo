Personalization API enables access to all user information we have obtained within the user interest modelling process. Personalisation API currently exposes the following interface:

Method Description
MP.me.getUrlsByTag() Returns URL list sorted by the relevancy of given tag.
MP.me.getTagsByUrl() Returns tag list sorted by the relevancy of given URL address.
MP.me.getDomains() Returns domain list sorted by the visit count.
MP.me.getUserTags() Returns weighted list of tags with the highest relevancy within the whole user model (global user interests).
MP.me.getUserTagsByDomain() Returns weighted list of tags with the highest relevancy within the given domain (local user interests).
MP.me.getTagRelevancy() Returns relevancy of given tag.
MP.me.getUrlDetails() Returns details for given URL address.

Methods

getUrlsByTag

MP.me.getUrlsByTag(query,callback);

Returns URL list sorted by the relevancy of given tag.

Parameters

query (object)

tag (string)

A keyword which describes returned URL addresses.

maxResults (optional integer)

The maximum number of results to retrieve. Defaults to 10.

callback (function)

This parameter should be a function in the following form:

function (array of WeightedUrl results) { ... }


getTagsByUrl

MP.me.getTagsByUrl(query,callback);

Returns tag list sorted by the relevancy of given URL address.

Parameters

query (object)

url (string)

A URL to get tags for.

maxResults (optional integer)

The maximum number of results to retrieve. Defaults to 10.

callback (function)

This parameter should be a function in the following form:

function (array of WeightedTag results) { ... }


getDomains

MP.me.getDomains(query,callback);

Returns domain list sorted by the visit count, where domain relevancy is its visit count.

Parameters

query (object)

maxResults (optional integer)

The maximum number of results to retrieve. Defaults to 10.

callback (function)

This parameter should be a function in the following form:

function (array of WeightedUrl results) { ... }


getUserTags

MP.me.getUserTags(query,callback);

Returns weighted list of tags with the highest relevancy within the whole user model (global user interests).

Parameters

query (object)

maxResults (optional integer)

The maximum number of results to retrieve. Defaults to 10.

callback (function)

This parameter should be a function in the following form:

function (array of WeightedTag results) { ... }


getUserTagsByDomain

MP.me.getUserTagsByDomain(query,callback);

Returns weighted list of tags with the highest relevancy within the given domain (local user interests).

Parameters

query (object)

domain (string)

A domain to get user tags for.

maxResults (optional integer)

The maximum number of results to retrieve. Defaults to 10.

callback (function)

This parameter should be a function in the following form:

function (array of WeightedTag results) { ... }


getTagRelevancy

MP.me.getTagRelevancy(query,callback);

Returns relevancy of given tag.

Parameters

query (object)

tag (string)

A name of tag to get relevancy of.

callback (function)

This parameter should be a function in the following form:

function (relevancy) { ... }

where relevancy is the number denoting relevancy of given tag.


getUrlDetails

MP.me.getUrlDetails(query,callback);

Returns details for given URL address.

Parameters

query (object)

url (string)

A URL address to get details for.

callback (function)

This parameter should be a function in the following form:

function (UrlDetails) { ... }


Types

WeightedUrl

(object)

url (string)

A URL address.

relevancy (number)

The relevancy of this URL address.


WeightedTag

(object)

tag (string)

A name of tag.

relevancy (number)

The relevancy of this tag.


UrlDetails

(object)

id (string)

The unique identifier for given URL.

url (string)

The URL navigated to by a user.

title (string)

The title of the page when it was last loaded.

lastVisitTime (string)

When this page was last loaded, represented in milliseconds since the epoch.

visitCount (string)

The number of times the user has navigated to this page.

typedCount (number)

The number of times the user has navigated to this page by typing in the address.
