Connect

MP.db.connect(channel, onmessage, onerror, onconnect, ondisconnect);

Connects to the specified communication channel and returns a WebSocket object. You can specify optional event handlers for all WebSocket events. Please see the WebSocket API reference for more details on how to work with it.

Parameters

channel (string)

Name of channel to connect into.

onmessage (optional function)

This parameter should be a function in the following form:

function (eventData) { var message=eventData.data; ... }

onerror (optional function)

This parameter should be a function in the following form:

function (eventData) { ... }

onconnect (optional function)

This parameter should be a function in the following form:

function (eventData) { ... }

ondisconnect (optional function)

This parameter should be a function in the following form:

function (eventData) { ... }

Return value

(WebSocket)

Please see the WebSocket API reference for more details on how to work with it.
